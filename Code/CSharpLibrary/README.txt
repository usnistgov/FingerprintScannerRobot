1.	Move: the superclass of PositionMove and SensorMove. Represents a move as specified in the Arduino - Computer Communication Protocol.
2.	PositionMove: represents a position move as specified in the Arduino - Computer Communication Protocol.
3.	SensorMove: represents a sensor move as specified in the Arduino - Computer Communication Protocol.
4.	NAFSTR: represents a robot. Contains methods for each of the robot's commands.
